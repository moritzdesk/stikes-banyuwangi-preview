## 🎬 Opsi 2: Quick Demo (Jika Ada Waktu)

### Setup Demo (15 menit):

**1. Daftar Sanity (2 menit):**
```bash
# Buka browser
https://www.sanity.io/create

# Daftar dengan email
# Pilih "Blog" template (untuk demo cepat)
```

**2. Setup Project (5 menit):**
```bash
# Install Sanity CLI
npm install -g @sanity/cli

# Login
sanity login

# Create project
sanity init --template blog --project-id YOUR_PROJECT_ID

# Start studio
cd YOUR_PROJECT
sanity start
```

**3. Demo Content (5 menit):**
- Create 1 artikel dengan AI
- Show SEO settings
- Show media upload
- Publish

**4. Show Result (3 menit):**
- Show article di Sanity Studio
- Show API response
- Show bagaimana website bisa fetch

## 💡 Tips Presentasi

### Do's:
- ✅ **Focus pada benefits** untuk institusi
- ✅ **Emphasize FREE** untuk start
- ✅ **Show ROI** - time saved, cost saved
- ✅ **Keep it simple** - jangan terlalu technical
- ✅ **Use visuals** - screenshots, diagrams

### Don'ts:
- ❌ **Jangan terlalu technical** - wakil rektor bukan developer
- ❌ **Jangan terlalu detail** - keep it high-level
- ❌ **Jangan compare terlalu banyak** - focus pada Sanity
- ❌ **Jangan promise terlalu banyak** - realistic expectations
