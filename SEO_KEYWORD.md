## ✍️ 2. Content Optimization

### Title Tag Optimization
- [ ] **Title dengan Keyword** - Title tag mengandung keyword utama
- [ ] **Title Length** - 50-60 karakter (optimal untuk display)
- [ ] **Title Uniqueness** - Setiap halaman memiliki title yang unik
- [ ] **Title Structure** - Struktur title yang konsisten
- [ ] **Title CTA** - Call-to-action di title jika relevan

### Meta Description Optimization
- [ ] **Description dengan Keyword** - Meta description mengandung keyword
- [ ] **Description Length** - 150-160 karakter (optimal untuk display)
- [ ] **Description CTA** - Call-to-action di description
- [ ] **Description Uniqueness** - Setiap halaman memiliki description yang unik
- [ ] **Description Appeal** - Description yang menarik dan clickable

### Heading Optimization
- [ ] **H1 dengan Keyword** - H1 tag mengandung keyword utama
- [ ] **H2-H6 dengan Keywords** - Heading mengandung keyword relevan
- [ ] **Heading Hierarchy** - Hierarki heading yang logis

- [ ] **Keyword in First Paragraph** - Keyword muncul di paragraph pertama
- [ ] **Natural Keyword Usage** - Keyword digunakan secara natural
- [ ] **Keyword Density** - Keyword density 1-2% (tidak over-optimization)
- [ ] **Semantic Keywords** - Menggunakan semantic keywords terkait
- [ ] **LSI Keywords** - Latent Semantic Indexing keywords
- [ ] **Content Length** - Minimal 300 kata, ideal 1000+ kata untuk artikel
- [ ] **Content Quality** - Konten original, informatif, dan bermanfaat
- [ ] **Content Uniqueness** - Konten original, bukan duplicate

### Internal Linking (Content-Based)

- [ ] **Internal Link Strategy** - Strategi internal linking
- [ ] **Link Context** - Link dalam konteks yang relevan

## 📝 Content Guidelines

### Writing Best Practices
- Tulis untuk manusia, bukan untuk search engine
- Gunakan keyword secara natural
- Fokus pada kualitas konten
- Pastikan konten original dan bermanfaat
- Update konten secara berkala

### Keyword Best Practices
- Jangan keyword stuffing
- Gunakan keyword dalam konteks yang relevan
- Variasi keyword untuk natural writing
- Fokus pada user intent
- Monitor dan adjust berdasarkan performa
