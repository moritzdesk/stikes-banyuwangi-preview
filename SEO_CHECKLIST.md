## 📋 SEO Standar (Traditional SEO)

### 1. On-Page SEO

#### Meta Tags
- [ ] **Title Tag** - Unik, deskriptif, 50-60 karakter, mengandung keyword utama
- [ ] **Meta Description** - Menarik, 150-160 karakter, mengandung CTA
- [ ] **Meta Keywords** - (Opsional, tidak terlalu penting lagi)
- [ ] **Open Graph Tags** - Untuk sharing di social media (Facebook, LinkedIn)
- [ ] **Twitter Card Tags** - Untuk sharing di Twitter
- [ ] **Canonical URL** - Mencegah duplicate content
- [ ] **Robots Meta Tag** - Kontrol indexing (index/noindex, follow/nofollow)
- [ ] **Viewport Meta Tag** - Untuk mobile responsiveness
- [ ] **Charset Meta Tag** - UTF-8 encoding

#### Heading Structure
- [ ] **H1 Tag** - Satu H1 per halaman, mengandung keyword utama
- [ ] **H2-H6 Tags** - Hierarki heading yang logis dan terstruktur
- [ ] **Semantic HTML** - Menggunakan tag HTML5 yang tepat (header, nav, article, section, footer)

#### Content Optimization
- [ ] **Keyword Research** - Riset keyword yang relevan untuk setiap halaman
- [ ] **Keyword Density** - 1-2% keyword density (natural, tidak keyword stuffing)
- [ ] **Content Quality** - Konten original, informatif, dan bermanfaat
- [ ] **Content Length** - Minimal 300 kata per halaman, ideal 1000+ kata untuk artikel
- [ ] **Internal Linking** - Link ke halaman relevan di dalam website
- [ ] **External Linking** - Link ke sumber terpercaya (authority sites)
- [ ] **Alt Text untuk Images** - Deskriptif, mengandung keyword jika relevan
- [ ] **Image Optimization** - Format WebP/AVIF, compressed, ukuran file kecil
- [ ] **Video Optimization** - Transcript, captions, thumbnail yang menarik

#### URL Structure
- [ ] **URL yang SEO-friendly** - Pendek, deskriptif, mengandung keyword
- [ ] **HTTPS** - SSL certificate terpasang
- [ ] **WWW vs Non-WWW** - Konsisten (pilih salah satu, redirect yang lain)
- [ ] **URL Slugs** - Menggunakan hyphens, lowercase, tanpa special characters

### 2. Technical SEO

#### Site Speed
- [ ] **Page Load Time** - < 3 detik (target < 2 detik)
- [ ] **Core Web Vitals** - LCP < 2.5s, FID < 100ms, CLS < 0.1
- [ ] **Image Optimization** - Lazy loading, responsive images
- [ ] **CSS/JS Minification** - File CSS dan JS di-minify
- [ ] **Caching** - Browser caching dan server-side caching
- [ ] **CDN** - Content Delivery Network untuk static assets
- [ ] **Gzip/Brotli Compression** - Kompresi file untuk transfer lebih cepat

#### Mobile Optimization
- [ ] **Responsive Design** - Website mobile-friendly
- [ ] **Mobile-First Indexing** - Optimasi untuk mobile-first
- [ ] **Touch-Friendly** - Button dan link mudah di-tap
- [ ] **Viewport Configuration** - Viewport meta tag yang benar
- [ ] **Mobile Page Speed** - Fast loading di mobile devices

#### Site Structure
- [ ] **XML Sitemap** - Sitemap.xml yang up-to-date
- [ ] **Robots.txt** - File robots.txt yang benar
- [ ] **Breadcrumbs** - Breadcrumb navigation dengan structured data
- [ ] **404 Page** - Custom 404 error page yang user-friendly
- [ ] **301 Redirects** - Redirect URL lama ke URL baru jika ada perubahan

#### Schema Markup (Structured Data)
- [ ] **Organization Schema** - Schema untuk institusi
- [ ] **BreadcrumbList Schema** - Untuk breadcrumbs
- [ ] **Article Schema** - Untuk artikel/blog posts
- [ ] **FAQ Schema** - Untuk FAQ section
- [ ] **LocalBusiness Schema** - Untuk informasi lokasi
- [ ] **Course Schema** - Untuk program studi/kursus
- [ ] **Event Schema** - Untuk acara/event
- [ ] **Person Schema** - Untuk pimpinan/dosen
- [ ] **Review Schema** - Untuk testimoni/review

### 3. Off-Page SEO

- [ ] **Backlinks** - Link dari website berkualitas tinggi
- [ ] **Social Media Presence** - Aktif di social media
- [ ] **Local Citations** - Listing di Google Business Profile, direktori lokal
- [ ] **Guest Posting** - Artikel di website lain dengan backlink
- [ ] **Directory Listings** - Listing di direktori pendidikan

### 4. Local SEO

- [ ] **Google Business Profile** - Setup dan optimasi lengkap
- [ ] **NAP Consistency** - Name, Address, Phone konsisten di semua platform
- [ ] **Local Keywords** - Keyword dengan lokasi (contoh: "Universitas Dr. Soekardjo Banyuwangi")
- [ ] **Local Content** - Konten yang relevan dengan lokasi
- [ ] **Local Reviews** - Review dari mahasiswa/alumni

### 5. Content Marketing

- [ ] **Blog/Artikel** - Konten berkualitas secara rutin
- [ ] **Content Calendar** - Jadwal publikasi konten
- [ ] **Content Variety** - Artikel, infografis, video, podcast
- [ ] **Evergreen Content** - Konten yang tetap relevan
- [ ] **Content Updates** - Update konten lama secara berkala
