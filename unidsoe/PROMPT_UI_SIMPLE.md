## 🚀 Cara Menggunakan

### Method 1: Buka File HTML Langsung

1. **Buka file:** `unidsoe/public/prompt-ui.html` di browser
2. **Token sudah terisi** (default)
3. **Input prompt** dan klik "🚀 Generate Response"
4. **Result muncul** - Bisa copy ke clipboard

### Method 2: Serve via Dev Server

**Jika dev server jalan:**
```
http://localhost:3333/prompt-ui.html
```
